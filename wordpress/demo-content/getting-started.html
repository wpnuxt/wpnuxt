This guide will help you get started with WPNuxt in your local development environment.

Prerequisites

Before you begin, make sure you have:

  • Node.js 20 or later
  • A WordPress site with WPGraphQL installed

Quick Start

Install the WPNuxt module in your Nuxt project:

pnpm add @wpnuxt/core

Then configure it in your nuxt.config.ts with your WordPress URL.
