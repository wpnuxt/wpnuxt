# Other Platforms

WPNuxt works on any platform that supports Nuxt 4.

## Netlify

```toml [netlify.toml]
[build]
  command = "pnpm build"
  publish = ".output/public"

[build.environment]
  WPNUXT_WORDPRESS_URL = "https://your-wordpress-site.com"
```

## Cloudflare Pages

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  nitro: {
    preset: 'cloudflare-pages'
  }
})
```

Set `WPNUXT_WORDPRESS_URL` in Cloudflare dashboard.

## Self-Hosted (Node.js)

Build and run:

```bash
pnpm build
node .output/server/index.mjs
```

Set environment variables:

```bash
WPNUXT_WORDPRESS_URL=https://your-wordpress-site.com node .output/server/index.mjs
```

## Docker

```dockerfile [Dockerfile]
FROM node:20-alpine

WORKDIR /app
COPY .output .output

ENV WPNUXT_WORDPRESS_URL=https://your-wordpress-site.com

EXPOSE 3000
CMD ["node", ".output/server/index.mjs"]
```

## Environment Variables

All platforms need:

| Variable | Required | Description |
|----------|----------|-------------|
| `WPNUXT_WORDPRESS_URL` | Yes | WordPress site URL |
| `WPNUXT_DEBUG` | No | Enable debug logs |
