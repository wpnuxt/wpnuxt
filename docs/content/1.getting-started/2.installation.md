## Requirements

- Nuxt 4.x
- Node.js 20+
- WordPress with [WPGraphQL](https://wordpress.org/plugins/wp-graphql/) plugin

## Install

::code-group
```bash [pnpm]
pnpm add @wpnuxt/core
```
```bash [npm]
npm install @wpnuxt/core
```
```bash [yarn]
yarn add @wpnuxt/core
```
::

## Configure

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@wpnuxt/core'],

  wpNuxt: {
    wordpressUrl: 'https://your-wordpress-site.com'
  }
})
```

Or use environment variables:

```bash [.env]
WPNUXT_WORDPRESS_URL=https://your-wordpress-site.com
```

## Verify

Run the dev server:

```bash
pnpm dev
```

If you see "WPNuxt module loaded" in the console, you're ready to go.
