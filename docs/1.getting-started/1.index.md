## Installation

Install the core module:

::code-group

```bash [pnpm]
pnpm add @wpnuxt/core
```

```bash [npm]
npm install @wpnuxt/core
```

```bash [yarn]
yarn add @wpnuxt/core
```

::

### Optional: Install Blocks Package

For rendering Gutenberg blocks as Vue components:

::code-group

```bash [pnpm]
pnpm add @wpnuxt/blocks
```

```bash [npm]
npm install @wpnuxt/blocks
```

::

## Setup

Add `@wpnuxt/core` to your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@wpnuxt/core'],

  wpNuxt: {
    wordpressUrl: 'https://your-wordpress-site.com',
    graphqlEndpoint: '/graphql',
    downloadSchema: true,
    debug: false,
    cache: {
      enabled: true,
      maxAge: 300,
      swr: true,
    },
  },
})
```

With blocks:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@wpnuxt/core', '@wpnuxt/blocks'],

  wpNuxt: {
    wordpressUrl: 'https://your-wordpress-site.com',
  },

  wpNuxtBlocks: {
    imageDomains: ['your-wordpress-site.com'],
  },
})
```

## Environment Variables

You can also configure WPNuxt using environment variables:

```bash [.env]
WPNUXT_WORDPRESS_URL=https://your-wordpress-site.com
WPNUXT_GRAPHQL_ENDPOINT=/graphql
WPNUXT_DOWNLOAD_SCHEMA=true
WPNUXT_DEBUG=false
```

## WordPress Requirements

### Required Plugins

1. **WPGraphQL** - Install and activate the [WPGraphQL](https://wordpress.org/plugins/wp-graphql/) plugin

For `@wpnuxt/blocks`:

2. **WPGraphQL Content Blocks** - Install [WPGraphQL Content Blocks](https://github.com/wpengine/wp-graphql-content-blocks)

### Recommended Settings

In WordPress Admin > GraphQL > Settings:

- **GraphQL Endpoint**: `/graphql` (default)
- **Enable GraphQL Introspection**: Yes (required for schema download)
- **Public Introspection**: Yes (during development)

## Basic Usage

```vue [pages/index.vue]
<script setup lang="ts">
const { data: posts } = await usePosts()
</script>

<template>
  <article v-for="post in posts" :key="post.id">
    <h2>{{ post.title }}</h2>
    <div v-sanitize-html="post.excerpt" />
  </article>
</template>
```
