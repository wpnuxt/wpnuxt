## Installation

Install the module:

::code-group

```bash [pnpm]
pnpm add wpnuxt
```

```bash [npm]
npm install wpnuxt
```

```bash [yarn]
yarn add wpnuxt
```

::

## Setup

Add `wpnuxt` to your `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ["wpnuxt"],

  wpNuxt: {
    wordpressUrl: "https://your-wordpress-site.com",
    graphqlEndpoint: "/graphql",
    downloadSchema: true,
    debug: false,
    cache: {
      enabled: true,
      maxAge: 300,
      swr: true,
    },
  },
});
```

## Environment Variables

You can also configure WPNuxt using environment variables:

```bash [.env]
WPNUXT_WORDPRESS_URL=https://your-wordpress-site.com
WPNUXT_GRAPHQL_ENDPOINT=/graphql
WPNUXT_DOWNLOAD_SCHEMA=true
WPNUXT_DEBUG=false
```

## WordPress Requirements

### Required Plugins

1. **WPGraphQL** - Install and activate the [WPGraphQL](https://wordpress.org/plugins/wp-graphql/) plugin

### Recommended Settings

In WordPress Admin → GraphQL → Settings:

- **GraphQL Endpoint**: `/graphql` (default)
- **Enable GraphQL Introspection**: Yes (required for schema download)
- **Public Introspection**: Yes (during development)
